About

Available packages

The tools available on this site only provide access to the code for packages that are currently part of Bioconductor. This means that they are either in the current release branch, or have been accepted during this release cycle and are available from the developmental version of Bioconductor. If your package is currently under review, but has not yet been accepted, it will not appear here. Likewise, deprecated packages from previous releases are also unavailable. It is also limited to "software" packages and doesn't provide access to "workflows", "books", "annotation", etc. This may change in the future.

Update frequency

The central Biocondutor Git repository can be found at git.bioconductor.org. The files backing this site are derived from a local mirror of the central repository. Updates are synced to the mirror automatically every 15 minutes. If you do not see code you recently commited to the main git repository, please wait at least 15 minutes before wondering why.

Technologies

  • The git browser is powered by GitList.
  • The code search functionality is driven by Zoekt.

All resources used to create the Docker containers that power the web services and mirror updates can be found on GitHub.

The details below related only to the subdomain code.bioconductor.org. This site is hosted and maintained independently of any other Bioconducutor websites and the following statements may not hold true for any other sites.

What We Collect and Receive

  • Contact us via email — for example, when you ask for support, send us questions or comments, or report a problem, we will collect your name, email address, message, etc. We use this data solely in connection with answering the queries we receive.
  • Usage data — when you visit our site, we will store: the website that you visited us from, the parts of our site you visit, the date and duration of your visit, your anonymised IP address, information from the device (device type, operating system, screen resolution, language, country you are located in, and web browser type) you used during your visit, and more. We process this usage data in Matomo Analytics.
  • Cookies — This site does not use cookies and no information is stored on your computer.

Why we collect data

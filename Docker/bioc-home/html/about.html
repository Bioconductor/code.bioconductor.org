Updates

The central Biocondutor Git repository can be found at git.bioconductor.org. The files backing this this site are derived from a local mirror of the central repository. Updates are synced to the mirror automatically every 30 minutes. If you do not see code you recently commited to the main git repository, please wait at least 30 minutes before wondering why.

Technologies

  • The git browser is powered by GitList.
  • The code search functionality is driven by Zoekt.

All resources used to create the Docker containers that power the web services and mirror updates can be found on GitHub.

Hosting

The site is currently hosted on a Kubernetes instance supported by the IT Services Team at the European Molecular Biology Laboratory.

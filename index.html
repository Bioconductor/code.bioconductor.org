Unofficial Bioconductor Code Tools

This site provides some experimental tools for exploring the code hosted in the Bioconductor git repository.

Excluded Packages

In the interest of saving space on this web hosting a small number of large packages have been excluded from the service. Any package where the directory size is greater than 150MB has been excluded. If a packe you're interested in is missing, click here to see if it is on the list of excluded packages.

Update frequency

The primary Bioconductor git repository is checked for new commits once an hour, on the hour. You can view a log of this process, with a time stamp for each package here.

Browse package sources

Gitlist lets us browse the master and release branches of the packages in the Bioconductor project.


Search codebase

It's sometimes desirable to be able search the contents of all Bioconductor packages, perhaps to see if anyone is using a function from you package, or to find examples of how other packages have used specific function. We use Zoekt to index and search of the master branch for all Bioconductor packages.
